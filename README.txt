# Invitación (2 páginas, sin parallax)

## Flujo
1) Idioma -> 2) Código (nombre) -> 3) Landing con mensaje + botón
4) Botón abre **invitation.html** (segunda página)

## Abrir
- Abre `index.html` (recomendado: Live Server en VS Code).

## Banner
- Reemplaza `assets/banner.jpg` por tu imagen (mismo nombre).

## Editar contenido
- Maquetación de la invitación: `invitation.html`
- Textos ES/IT: `script.js` (landing) y `invitation.js` (labels en invitación)


## Códigos permitidos
- Edita `codes.js` y añade tus invitados.
- Si alguien escribe un código que NO está en la lista, verá un error.
